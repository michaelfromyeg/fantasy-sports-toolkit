---
name: waiver-wire
description: Evaluate available free agents and recommend waiver claims with a FAAB bid for each. Works with any connected fantasy data provider.
---

When asked about the waiver wire:

1. Identify the manager's roster needs (thin positions, injured starters, upcoming byes).
2. Get the available free agents and their recent usage trend from whatever fantasy data provider
   MCP is connected (Sleeper, ESPN, Yahoo, etc.). The signal that matters most is a player whose
   snap share, target share, or touches just jumped. If no provider is connected, ask the manager
   for the free-agent list.
   - Confirm the opportunity with the bundled provider-independent MCPs: `reddit`
     (r/fantasyfootball waiver threads, beat-reporter news) and `web-fetch` for depth charts
     or injury reports, plus `sports-data` for the player's team context.
3. Rank pickups by expected rest-of-season value for THIS roster, not raw talent.
4. For each recommended claim, give a FAAB bid as a percent of the remaining budget:
   - League-winner / clear new starter: 25-50%.
   - Useful flex or handcuff to a star: 5-15%.
   - Speculative stash: 0-3%.
5. Name the bench player to drop for each add.

Prioritize opportunity (role and volume) over name recognition. A backup who just inherited the
lead-back job beats a famous name buried on the depth chart. The valuation logic is the same
whichever provider supplies the data.
